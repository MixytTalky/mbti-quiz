## 功能規劃

### 功能 1：開始頁面
- **輸入**：無（使用者點擊「開始測驗」）
- **處理**：顯示工具標題、簡介、開始按鈕
- **輸出**：進入第 1 題
- **邊界條件**：無

### 功能 2：答題流程（24 題）
- **輸入**：使用者在每題選擇一個選項（2 選 1）
- **處理**：每題對應四維度之一，選擇 A 或 B 分別對該維度的兩端加分
- **輸出**：
  - 顯示當前題號 / 總題數（如 5/24）
  - 顯示進度條（以柔和漸層填充）
  - 選擇後自動滑入下一題（動畫過渡）
- **邊界條件**：不可跳題；可回到上一題修改答案

### 功能 3：結果頁面
- **輸入**：所有 24 題作答完成
- **處理**：各維度取分數較高的一端，組合成 4 字母人格（如 INFP）
- **輸出**：
  - MBTI 類型 + 暱稱（如「INFP — 夢想治癒師」）
  - 人格形象圖（水彩風）
  - 性格描述（3-5 行）
  - 四維度雷達圖（視覺化比例）
  - 全球占比 + 稀有度標籤（如「全球僅 1.5% SSR」）
  - 相性分析：「最佳拍檔」與「最衝類型」
  - 「分享結果」按鈕（複製文字 + 連結）
  - 「重新測驗」按鈕
- **邊界條件**：分享連結應附帶 UTM 參數

### 功能 4：社群分享
- **輸入**：使用者點擊「分享結果」
- **處理**：生成分享文字（含 MBTI 類型 + 暱稱 + 稀有度 + 工具連結 + UTM）
- **輸出**：複製到剪貼簿 + toast 提示「已複製！」

## 題目設計（24 題，每維度 6 題）

### E/I 維度（外向 vs 內向）

**第 1 題：週末到了，你比較想怎麼度過？**
- A. 約朋友出去聚會、逛街、唱歌 → E+1
- B. 待在家看書、追劇、享受獨處時光 → I+1

**第 2 題：在一個課堂或會議上，你通常是？**
- A. 主動發言、分享想法的人 → E+1
- B. 安靜聆聽，想好了再說的人 → I+1

**第 3 題：遇到問題時，你的第一反應是？**
- A. 找朋友或同事討論，聊聊就有靈感 → E+1
- B. 自己先想清楚，再決定要不要問人 → I+1

**第 4 題：在派對上，你通常？**
- A. 到處認識新朋友，越聊越興奮 → E+1
- B. 找幾個熟人聊天，或找個安靜角落 → I+1

**第 5 題：別人形容你時，更常說你？**
- A. 開朗活潑、很容易親近 → E+1
- B. 沉穩安靜、有點神秘 → I+1

**第 6 題：長時間社交後，你的感受是？**
- A. 充滿能量，覺得很開心 → E+1
- B. 需要獨處充電，有點累了 → I+1

### S/N 維度（實感 vs 直覺）

**第 7 題：你在看一幅畫時，會先注意到？**
- A. 畫的技法、色彩搭配、構圖細節 → S+1
- B. 畫想傳達的意境、故事和象徵意義 → N+1

**第 8 題：計劃旅行時，你更傾向？**
- A. 做好詳細攻略，景點、交通、預算都安排好 → S+1
- B. 只訂大方向，到了再隨興探索 → N+1

**第 9 題：老師或主管交代任務時，你希望？**
- A. 有清楚的步驟和具體指示 → S+1
- B. 只給方向和目標，讓我自己發揮 → N+1

**第 10 題：你更欣賞哪種人？**
- A. 腳踏實地、經驗豐富的務實者 → S+1
- B. 天馬行空、充滿遠見的夢想家 → N+1

**第 11 題：學習新事物時，你更喜歡？**
- A. 從具體的例子和實際操作開始 → S+1
- B. 先理解整體概念和理論框架 → N+1

**第 12 題：你覺得自己更擅長？**
- A. 處理眼前的事，把細節做到完美 → S+1
- B. 預見未來趨勢，構思各種可能性 → N+1

### T/F 維度（思考 vs 情感）

**第 13 題：好朋友做了一個你覺得錯誤的決定，你會？**
- A. 直接指出問題，真正的朋友不需要拐彎抹角 → T+1
- B. 先關心對方的感受，再委婉地給建議 → F+1

**第 14 題：做重大決定時，你更依賴？**
- A. 理性分析：列出利弊、數據、邏輯 → T+1
- B. 內心感受：這個選擇讓不讓我舒服 → F+1

**第 15 題：在團隊合作中發生爭執，你認為最重要的是？**
- A. 找出最正確、最有效率的解決方案 → T+1
- B. 確保每個人的感受都被照顧到 → F+1

**第 16 題：看完一部電影，你傾向？**
- A. 分析劇情邏輯、角色合理性 → T+1
- B. 回味情感共鳴，討論最感動的情節 → F+1

**第 17 題：你覺得「公平」是？**
- A. 每個人按照規則一視同仁 → T+1
- B. 考慮個別狀況，給予不同的照顧 → F+1

**第 18 題：收到批評時，你的反應是？**
- A. 客觀檢視批評有沒有道理，有幫助就改 → T+1
- B. 先在意對方的態度，然後才能聽進內容 → F+1

### J/P 維度（判斷 vs 感知）

**第 19 題：你的書桌或房間通常是？**
- A. 整齊有序，每樣東西都有固定位置 → J+1
- B. 有點亂但我找得到東西，這是我的風格 → P+1

**第 20 題：面對截止日期，你通常？**
- A. 提早完成，不喜歡拖到最後 → J+1
- B. 截止前才靈感爆發，最後衝刺完成 → P+1

**第 21 題：計劃改變時，你的反應是？**
- A. 有點不安，我喜歡照計劃走 → J+1
- B. 沒關係，反正隨機應變更有趣 → P+1

**第 22 題：你更喜歡哪種工作方式？**
- A. 列好待辦清單，一項一項勾掉 → J+1
- B. 看心情和狀態決定先做什麼 → P+1

**第 23 題：買東西時，你通常？**
- A. 先做功課比較，確定需要才買 → J+1
- B. 看到喜歡就買，享受隨興的快樂 → P+1

**第 24 題：對於生活，你更嚮往？**
- A. 穩定有規律的節奏，讓人安心 → J+1
- B. 充滿變化和驚喜，每天都不一樣 → P+1

## 技術備註

- 計分邏輯：24 題二選一，每題為四維度之一，A/B 分別加分給該維度兩端。完成後每維度取較高端，組合成 4 字母 MBTI

- 稀有度顯示：結果頁顯示全球占比百分比 + 稀有度徽章（SSR/SR/R/N），分享文字也包含稀有度
- 動畫過渡：選擇答案後使用 CSS transition 滑入下一題
- 無後端依賴：所有邏輯前端完成
- 深色主題：預設開啟，使用 UIUX skill 的 CSS tokens
- 零 emoji：所有 UI 圖示使用 AI 生成圖片，禁止使用原生 emoji
